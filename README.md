# SQL Business Performance Analysis

## Project Overview

This project demonstrates how SQL can be used to solve real-world business problems by analyzing customer behavior, revenue performance, acquisition channels, and product trends.

The objective was to identify key business drivers and generate actionable insights through data analysis.

---

## Tools Used

- SQL
- Joins
- Aggregations
- CASE WHEN
- GROUP BY
- KPI Analysis
- Business Intelligence

---

## Business Questions Solved

### Revenue Analysis
- Revenue by acquisition channel
- Average revenue per user
- Channel effectiveness comparison

### Customer Analysis
- User growth and decline patterns
- Purchase frequency trends
- Customer contribution to revenue

### Product Analysis
- Top performing products
- Declining products
- Product growth classification

### Business Performance Analysis
- Revenue decline drivers
- Country-wise performance analysis
- Root cause investigation
